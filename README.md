# 🏠 Room4Rent

<p align="center">
  <strong>Find, List & Rent Properties Easily.</strong>
</p>

<p align="center">
  A modern property rental platform built with Next.js where users can browse, list, and manage rental properties.
</p>

---

## 🌐 Live Demo

https://room4rent2.vercel.app

---

## ✨ Features

- 🏠 Browse Rental Properties
- 🔍 Search & Filter Properties
- 📍 Location-Based Listings
- 💰 Property Price Details
- 🛏️ Property Information (Bedrooms, Bathrooms, Area)
- 📷 Multiple Property Images
- ❤️ Save Favorite Properties
- 👤 User Authentication
- 📝 List Your Own Property
- 📱 Fully Responsive Design
- ⚡ Fast Performance
- 🎨 Clean & Modern UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/nikhilaiml/room4rent2.git
```

Move into the project

```bash
cd room4rent2
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env.local` file.

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```
